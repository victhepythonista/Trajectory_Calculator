A simple GUI app I made to help me during plotting trajectories for sprites in pygame
Its usage are described below



	[PLATTFORMS]

-Windows


		[USAGE]
 
 run the application
 
 follow the commands and navigate to the graphbook screen
 
 draw on the space with your mouse  and press
    S - to save it to a randomly named file in Desktop/trajectoies  folder
    F - to save it to a user named file in Desktop/trajectoies  folder
